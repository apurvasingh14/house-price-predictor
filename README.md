California Housing Data Analysis
Dataset Description

The California Housing dataset contains information about housing districts in California. The dataset includes 20,640 samples and 8 numerical features such as median income, house age, average rooms, and location coordinates.

Exploratory Data Analysis

Exploratory Data Analysis (EDA) was performed to understand the dataset. Summary statistics and visualizations were used to analyze distributions, correlations, and patterns within the data.
